Enter your Personal Access Token to get started

How to get a token:

  1. Go to GitHub Settings → Developer settings → Personal access tokens
  2. Click "Generate new token (classic)"
  3. Select scopes:
    repo (full control - needed for write access)
    read:user (read user profile)
  4. Copy the generated token and paste above
Important: The repo scope (full control) is required for write access to update README files in your repositories.
